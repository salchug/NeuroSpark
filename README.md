# NeuroSpark

NeuroSpark is an iOS application designed to support early autism screening and guidance by combining modern mobile development with AI-driven analysis.

The app collects structured child information and behavioral inputs, analyzes short video data using machine learning techniques, and provides personalized insights and recommendations to assist parents and caregivers. NeuroSpark focuses on accessibility, clarity, and user-centered design, aiming to make early awareness tools more approachable through mobile technology.

Tech stack & tools used:
- Swift & UIKit (programmatic UI, no Storyboard)
- Xcode
- Firebase (Authentication, real-time data handling)
- Machine Learning / AI models for behavioral analysis
- RESTful data flow & modular architecture
- iOS Human Interface Guidelines for UX design
